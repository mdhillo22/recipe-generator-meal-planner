# Coding Prompts - Recipe Generator & Meal Planner

## FastAPI Backend Implementation

**Prompt**: "Write a FastAPI application for a recipe generator and meal planner that:
1. Has endpoints to generate recipes from ingredients using together.ai
2. Supports dietary preferences and cuisine types
3. Has endpoints to generate multi-day meal plans
4. Calculates nutrition information from recipes
5. Stores recipes and meal plans in memory
6. Serves a modern web frontend

Include proper error handling, CORS middleware, and Pydantic models."

## Frontend JavaScript Implementation

**Prompt**: "Write JavaScript for a single-page application that:
1. Handles tabbed navigation between recipe and meal plan sections
2. Submits recipe generation requests with ingredients and preferences
3. Displays recipes with formatted ingredients, instructions, and nutrition
4. Submits meal plan generation requests
5. Displays meal plans in a calendar/grid layout with daily summaries
6. Handles loading states and errors

Use vanilla JavaScript with clean, maintainable code."

## Nutrition Calculation

**Prompt**: "Write Python code to calculate nutrition information from recipe ingredients. Create a basic nutrition database for common ingredients and implement a function that estimates calories, protein, carbs, and fat per serving based on the ingredients list."

## LLM Prompt Templates

**Prompt**: "Create prompt templates for:
1. Recipe generation - instructs LLM to create detailed recipes from ingredients with instructions, nutrition estimates, and dietary considerations
2. Meal planning - instructs LLM to create multi-day meal plans with variety and nutrition targets

The templates should request structured JSON responses with all necessary recipe and meal plan information."
